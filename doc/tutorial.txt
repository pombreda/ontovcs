1. Download the latest version.
2. Unpack anywhere.
3. Add `bin' directory to your PATH.
   If you are on Windows, `setup.cmd' will do this for you.
4. Type `owl2diff'.
   You should see the usage hint.
5. Create a repository and CD to it.
   hg init ontovcstest; cd ontovcstest
   OR
   git init ontovcstest; cd ontovcstest
6. Type `owl2enable'.
   This will enable `owl2diff' and `owl2merge' for this repository.
7. Create an *.owl file and add it.
   For testing purposes there are some owl-files in the `sample' directory.
   Just copy one of them to the repository and rename it to `pizza.owl'.
   Then type `hg add pizza.owl' or `git add pizza.owl'
8. Commit
   hg commit -m "pizza.owl added"
   git commit pizza.owl -m "pizza.owl added"
9. Change one of *.owl files in the repository.
   For example, replace the pizza.owl with another pizza-*.owl from the `samples' directory.
10. Type `hg extdiff' of `git difftool'.
    If everything is ok, you will see the changes summary,
	then full changes in functional syntax, each prefixed with `+' or `-':
	
Ontology format changed to: OWL Functional Syntax
Modified:

    NamedIndividual: <http://www.co-ode.org/ontologies/pizza/pizza.owl#Germany>
