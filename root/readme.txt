This is OntoVCS version @build.version@

OntoVCS is a collaborative ontology development toolset integrated with source code version control systems.

Requirements
============

 * Java 1.6 or higher and `java` in `PATH`;
   http://www.java.com/download/
 * Java SWT library for your OS. Put `swt.jar` into `lib/`;
   http://www.eclipse.org/swt/
 * Git or Mercurial and a corresponding command-line client (`git` or `hg`) in `PATH`;
   http://git-scm.com/
   http://mercurial.selenic.com/
 * OWL Functional Syntax familiarity;
   http://www.w3.org/TR/owl2-syntax/

=====
