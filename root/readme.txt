This is OntoVCS version @build.version@

OntoVCS is a collaborative Semantic Web ontology development toolset
which integrates with source code version control systems.

Requirements
============

 * Java 1.6 or higher and `java` in `PATH`;
   http://www.java.com/download/
 * Java SWT library for your OS. Put `swt.jar` into `lib/swt/` directory;
   http://www.eclipse.org/swt/
 * Git or Mercurial and a corresponding command-line client (`git` or `hg`) in `PATH`;
   http://git-scm.com/
   http://mercurial.selenic.com/

Usage
=====

See the tutorial at http://code.google.com/p/ontovcs/wiki/Tutorial
